decimalSubtract(number1, number2) Decimal

This function accepts two parameters (number1, number2). number2 is subtracted from number1.

Code

                var standardResult = 456.456 - 456.123; 		// standardResult should return 0.333 but it doesn't :(
                var decimalResult = decimalSubtract(456.456, 456.123);	// decimalResult will return 0.333 :)
